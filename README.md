# Personal Portfolio Website - Vaila Ravi Shankar

This is the source code for my personal portfolio website, built with **React**, **TypeScript**, **Vite**, and **Vanilla CSS**.

## Live Demo
Check out the live version here: [https://Ravi20051.github.io/](https://Ravi20051.github.io/)

## Features
- **Data Science Playground**: An interactive Least-Squares Linear Regression simulator that computes and plots fit lines and $R^2$ metrics dynamically.
- **Glassmorphic UI**: Premium dark mode aesthetics with responsive grid elements, neon glow animations, and CSS custom variables.
- **Timeline Resume**: A structured overview of education, projects, skills, and certifications, fully optimized for printing/PDF conversion.
- **Contact Center**: Interactive form submission with validation and status feedback.

## Tech Stack
- **Frontend**: React 19, TypeScript
- **Styling**: Vanilla CSS (CSS variables, flexbox, grid, keyframe animations)
- **Icons**: Lucide React
- **Bundler**: Vite
- **Deployment**: GitHub Actions & GitHub Pages
